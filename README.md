# Zhiva Builder

Zhiva app builder and packager.

## Installation

```bash
ingr Zhiva-builder
```

or

```bash
git clone https://github.com/wxn0brP/Zhiva-builder.git
cd Zhiva-builder
bun install
# Add to $PATH the src/index.ts file as zhiva-builder.
```
